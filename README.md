Cryptography CLI for AES (Advanced Encryption Standard) with CBC (Cipher Block Chaining).

## Libraries Used

- **soft-aes**: internal implementation of AES algorithm;
- **cliclack**: fancy CLI;
- **rand**: generate random keys and IVs (Initialization Vectors).

## Compilation

+ Install [Rust](https://rust-lang.org/tools/install/);
+ Run `cargo run` inside the root directory.
